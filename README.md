# Outils et jeux développés par Joé Laroche 

Il s'agit ici d'un projet personnel permettant grâce au signet safari, d'avoir une application sur téléphone stockant différents jeux et outils que j'apprécie et dont j'ai besoin.
À l'heure actuelle, voici les différentes applications : 
- qui-a-dit.html = jeu d'ambiance où les participants doivent deviner les réponses des autres joueurs, en sachant que tout le monde doit se défendre des différentes accusations pour ne pas perdre de points. Le jeu a été inventé par un créateur de contenu instagram : @le_double_jeu ;
- compteurv2.html = outil permettant de compter les points des joueurs. Par exemple, lors d'une partie de Flip7, cela peut être fatiguant/encombrant d'avoir de quoi écrire, etc. > cet outil est une solution moins contraignante. + possibilité de compter les points spécialement pour le Yams (compteur.html était la première version)e
- croquettes = compteur de croquettes (permettant de savoir quelle quantité vous avez mis à votre animal/jour
